    **Vibration bug fix — draggable during shake:**

    The current shake uses `window.setFrameOrigin` which fights with window dragging. Instead of fighting, pause the shake during drag:
    1. In BuzzService, add `func pauseVibration()` and `func resumeVibration()` — pause cancels remaining shake work items but keeps `isVibrating` true and stores the remaining count.
    2. Actually simpler: since the click-to-stop already allows cancellation, and macOS window dragging naturally takes precedence over setFrameOrigin calls (the window manager handles drag), test if the existing DispatchWorkItem approach already allows dragging.

    Alternative simpler approach: In the vibrate function, before each setFrameOrigin call, check if the window is being dragged (NSEvent.pressedMouseButtons has bit 0 set). If so, skip that shake frame. After all shakes complete, reset to the CURRENT origin (not the captured one), since the user may have dragged during vibration.

    ```swift
    for i in 0..<shakes {
        let item = DispatchWorkItem { [weak self] in
            guard let self, self.isVibrating else { return }
            // Skip shake frame if user is dragging
            guard NSEvent.pressedMouseButtons & 1 == 0 else { return }
            let dx: CGFloat = (i % 2 == 0) ? distance : -distance
            window.setFrameOrigin(NSPoint(x: origin.x + dx, y: origin.y))
        }
        shakeWorkItems.append(item)
        DispatchQueue.main.asyncAfter(deadline: .now() + interval * Double(i), execute: item)
    }
    // Reset to current position (may have moved if dragged)
    let resetItem = DispatchWorkItem { [weak self] in
        guard let self else { return }
        self.isVibrating = false
        self.shakeWorkItems.removeAll()
        // Use current frame origin in case window was dragged
        let current = window.frame.origin
        let snapped = NSPoint(x: round(current.x), y: round(current.y))
        window.setFrameOrigin(snapped)
    }
    shakeWorkItems.append(resetItem)
    DispatchQueue.main.asyncAfter(deadline: .now() + total, execute: resetItem)
    ```
