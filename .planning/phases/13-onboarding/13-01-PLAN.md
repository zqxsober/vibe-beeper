    **ClaudeDetector.swift** — Create a `struct ClaudeDetector` with:
    - `static var claudeDirExists: Bool` — checks `FileManager.default.fileExists(atPath: NSHomeDirectory() + "/.claude")`
    - `static var claudeBinaryPath: String?` — searches these paths in order:
      1. `NSHomeDirectory() + "/.local/bin/claude"` (npm global install — most common)
      2. `/opt/homebrew/bin/claude` (Homebrew Apple Silicon)
      3. `/usr/local/bin/claude` (Homebrew Intel)
      4. Glob `NSHomeDirectory() + "/.nvm/versions/node/"` subdirectories for `*/bin/claude` (nvm installs)
    - `static var isInstalled: Bool` — returns `claudeBinaryPath != nil || claudeDirExists`
    - For nvm glob: use `FileManager.default.contentsOfDirectory(atPath:)` on `~/.nvm/versions/node/` and check each `{version}/bin/claude`

    **HookInstaller.swift** — Create a `struct HookInstaller` that replicates `setup.py` entirely in Swift. Do NOT spawn a Python subprocess. Static properties/methods:
    - `static let hooksDir = NSHomeDirectory() + "/.claude/hooks"`
    - `static let ipcDir = NSHomeDirectory() + "/.claude/claumagotchi"`
    - `static let settingsPath = NSHomeDirectory() + "/.claude/settings.json"`
    - `static let hookScript = hooksDir + "/claumagotchi-hook.py"`
    - `static let appPathFile = hooksDir + "/claumagotchi-app-path"`
    - `static var isInstalled: Bool` — checks hookScript file exists AND settings.json contains at least one hook entry with "claumagotchi-hook.py" in the command string. Use JSONSerialization to parse settings.json, iterate hooks dict, check for command containing "claumagotchi-hook.py".
    - `static func install() throws` — performs these steps:
      1. Create hooksDir and ipcDir with `withIntermediateDirectories: true`
      2. Set ipcDir permissions to 0o700
      3. Copy hook script from `Bundle.main.path(forResource: "claumagotchi-hook", ofType: "py")` to hookScript path. Set permissions to 0o755.
      4. Write `Bundle.main.bundlePath + "\n"` to appPathFile
      5. Read existing settings.json (or start with empty dict)
      6. For each of the 8 events (PreToolUse/5, PostToolUse/5, PostToolUseFailure/5, PermissionRequest/60, Notification/5, Stop/5, SessionStart/10, SessionEnd/5): remove any existing claumagotchi entries, then append new entry with `{"matcher": "", "hooks": [{"type": "command", "command": "python3 {hookScript}", "timeout": N}]}`
      7. Write settings back with `.prettyPrinted` and `.sortedKeys`
    - `enum InstallError: LocalizedError` with case `hookScriptNotInBundle`

    Import `Foundation` only. Both types must be `import Foundation` at the top, no AppKit or SwiftUI needed.

  <action>
    **AppMover.swift** — Create `import AppKit` at top, then `struct AppMover` with one static method:

    ```swift
    static func moveToApplicationsIfNeeded()
    ```

    Logic:
    1. Get `Bundle.main.bundlePath` as `currentPath`
    2. If `currentPath.hasPrefix("/Applications/")` — already there, return
    3. If `currentPath.contains(".build/")` — running from Xcode/SPM build, return
    4. If `currentPath.contains("AppTranslocation")` — show NSAlert with messageText "Move CC-Beeper to Applications" and informativeText "macOS is running CC-Beeper from a temporary location. Please drag CC-Beeper to your Applications folder and reopen it." with single "OK" button. Return after dismissal (do NOT attempt move).
    5. Otherwise: show NSAlert with messageText "Move CC-Beeper to Applications?" and informativeText "CC-Beeper works best when installed in your Applications folder." Two buttons: "Move to Applications" and "Don't Move".
    6. If user clicks "Move to Applications":
       - Compute `targetPath = "/Applications/" + (currentPath as NSString).lastPathComponent`
       - If targetPath exists, `try fm.removeItem(atPath: targetPath)`
       - Try `fm.copyItem(atPath: currentPath, toPath: targetPath)` first (works cross-volume from DMG). If that succeeds, `try? fm.removeItem(atPath: currentPath)`.
       - If copyItem fails, fall through silently (app still works from current location).
       - On success: `NSWorkspace.shared.openApplication(at: URL(fileURLWithPath: targetPath), configuration: NSWorkspace.OpenConfiguration())` then `NSApp.terminate(nil)`
    7. If user clicks "Don't Move" — return, proceed normally.
